### ***Installing Type Packages:***
- Run npm init -y to confirm all the default choices so that you can get a "package.json" files
- Inside package.json file, set "type":"module" so that you can use import in your project.
- Package to install is "@types/node --save-dev" it help TypeScript understand these Node.js APIs
